# Aquagreen
Ce container permet de gérer les commande de l'arrosage automatique
## Installation
Avant de déployer le container, il faut créer :
- un reseau externe frontend
- le fichier .env avec sont contenu ou la variable de la fichier docker-compose.yml
## Variable d'environnement
|Variable|Definition|
|---|----|
|MONGOURL|l'Url vers du serveur mongo (mongodb://mongo:27017)|
